# GDSquarefall

Ein minimalistisches Survival-Spiel, entwickelt mit Python und Pygame. Es dient als Einstiegsprojekt, um die Kernkonzepte der Spielelogik und Kollisionserkennung zu demonstrieren.

## 🚀 Features
- **Progressive Difficulty:** Die Fallgeschwindigkeit der Hindernisse erhöht sich basierend auf dem Score.
- **Collision Logic:** Eigenständige Implementierung der AABB-Kollisionsprüfung (Axis-Aligned Bounding Box).
- **GreenData UI:** Minimalistisches Design in Schwarz und Terminal-Grün.

## 🛠️ Tech-Stack
- **Sprache:** Python 3.12
- **Library:** Pygame

## 🎮 Steuerung
- **Bewegen:** Pfeiltasten (Links/Rechts)
- **Ziel:** Weiche den roten Quadraten so lange wie möglich aus!
